collectivité

https://rdafr.fr/Elements/c/C100005

Agent identifié comme étant une organisation ou une partie d'organisation, ou un groupe de personnes et/ou d'organisations. Une collectivité n’est considérée comme telle que si elle est identifiée par un nom particulier et si elle agit, ou peut agir, comme une unité. Un nom particulier consiste en une appellation spécifique plutôt qu’en une description générale.

  • S'applique à : rdafrc:C100005
  • Hérite de : agent
  • Type de noeud : IRI
  • Shape fermée
